Clock Class

Defined in: noname\clock.js:1

Tracking time within a game, synchronize animations and manage time related events like setTimeout and setInterval. All clocks are updated by the clockManager who has a 'master' clock itself to sync created clocks. All entities are synchronizez with the master clock untill you change it. You should not directly instantiate this class because it will not added to The clockManager pool and therefore will not be updated. Use the add method on the ClockManager instead.
