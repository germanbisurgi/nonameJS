TimeManager Class

Defined in: noname\time\timeManager.js:1

Manages all the game game clocks. Per default only one clock, the "Master Clock" is created and added to this manager as a property (not in its pool). the master clock synchronizes a lot of game activities like sprite animations, timers, ecc.If you want to apply a different clock with different properties you have to create a new clock for that. For example if you want that only entities of a certain type move in slow motion you must to create a new clock add it to this entities and change the motion property of that clock. New clocks should always be created with the clock manager create method otherwise it will be not updated.
